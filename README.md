# Miner-Game
礦世奇汖(miner pi)

運用C#程式語言和Unity遊戲引擎開發出跨三平台(Windows、Android、IOS)的模擬礦工採礦遊戲，主打亮點遊戲場景除了有經典的礦坑，還天馬行空創造了大海和天空場景，增加遊戲畫面的豐富度。
以下是我撰寫的主要功能：
1.	完成所有關卡的設計製做，設計了許多特殊得分規則、關卡失敗與成功觸發條件。
2.	可以自由選擇喜歡的遊戲場景進行遊戲，每個場景都藏有不同的陷阱，增加遊戲困難度。
3.	完成準備上架到Google play的所有相關工作，設定憑證、打包APK等工作。
4.	優化遊戲執行效能。

APK檔下載後可直接體驗遊戲。
